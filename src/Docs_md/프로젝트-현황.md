## 프로젝트 개요

**JobMoa**는 취업 지원 센터의 참여자 관리 및 취업 알선을 위한 통합 관리 시스템입니다.

- **프로젝트명**: TestProject (JobMoa)
- **버전**: 0.0.1-SNAPSHOT
- **패키징**: WAR
- **Java 버전**: 17
- **빌드 도구**: Maven
- **최근 마이그레이션**: Java Config 마이그레이션 완료 (2025-11-11)

---

## 기술 스택

### Backend
- **Framework**: Spring Boot 3.4.1 + Spring Framework 6.2.1
- **아키텍처**: Java Config 기반 (XML 설정 완전 제거)
- **ORM**: MyBatis 3.5.6
- **Database**: MS SQL Server
- **Connection Pool**: Apache Commons DBCP2
- **WebSocket**: Spring WebSocket + STOMP
- **AOP**: AspectJ
- **캐시**: Redis (Jedis, Lettuce)
- **메일**: Spring Mail

### Frontend
- **View Template**: JSP (Jakarta Servlet JSP 3.0)
- **CSS Framework**: AdminLTE 3.x
- **차트 라이브러리**: Chart.js, ApexCharts
- **JavaScript**: Vanilla JS (ES6+)
- **AJAX**: Fetch API, XMLHttpRequest

### Infrastructure
- **Database Server**: MS SQL Server (211.115.206.107:1433)
- **Redis Server**: 172.233.76.200:6379
- **Mail Server**: outbound.daouoffice.com:465

## 데이터베이스 구조

### 주요 테이블

#### 1. J_참여자관리
참여자의 전체 정보를 관리하는 메인 테이블

**주요 컬럼**
- `구직번호` (PK): 자동 증가 ID
- `참여자`: 참여자 이름
- `생년월일`, `성별`
- `전담자_계정`: 담당 상담사 계정
- `지점`: 소속 지점
- `등록일`, `최근상담일`, `초기상담일`
- `진행단계`: 상담 진행 단계
- `희망직무`, `희망급여`
- `취창업일`, `취업처`, `임금`, `직무`
- `취업유형`, `취업인센티브_구분`
- `마감`: 종료 여부 (bit)
- `메모`, `기타`

#### 2. J_참여자관리_로그인정보
상담사 및 관리자 계정 정보

**주요 컬럼**
- `전담자번호` (PK)
- `이름`, `아이디`, `비밀번호`
- `지점`, `권한`, `관리자권한`
- `이메일`, `전화번호`
- 실적 통계 (금일/금주/금월/금년 일반취업/알선취업)
- `입사일`, `최종발령일`

#### 3. J_참여자관리_자격증
참여자별 자격증 정보

**주요 컬럼**
- `자격증번호` (PK)
- `구직번호` (FK)
- `자격증`: 자격증 명칭

### 뷰 (Views)
- `CreateViewSQL.sql`에 정의된 통계 및 조회용 뷰

### 트리거 (Triggers)
- `CreateTriggersSQL.sql`에 정의된 자동화 트리거

## 주요 기능

### 1. 인증 및 권한 관리
- 로그인/로그아웃
- 세션 기반 인증 (6시간)
- 권한별 접근 제어 (상담, 관리자, 지점장)
- 로그인 인터셉터를 통한 보안 강화

### 2. 대시보드

### 3. 참여자 관리
- **CRUD 기능**: 참여자 등록, 조회, 수정, 삭제

- **자격증 관리**: 자격증 정보 등록
- **취업 정보 관리**: 취업 현황 및 추적
- **검색 및 필터링**: 다양한 조건 검색
- **페이지네이션**: 대량 데이터 처리

### 4. 취업 알선
- 구인 기업 정보 관리

### 5. 참여자 무작위 배정

```
    ↓
Persistence Layer (DAO, MyBatis)
    ↓
Database (MS SQL Server)
```
