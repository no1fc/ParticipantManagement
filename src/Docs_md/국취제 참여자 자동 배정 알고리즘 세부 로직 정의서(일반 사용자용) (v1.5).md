# 국취제 참여자 자동 배정 알고리즘 세부 로직 정의서(일반 사용자용) (v1.5)

## 1. 목적
국취제 참여자 자동 배정 화면의 사용 방법과 산정식(가중치) 의미를 일반 사용자 관점에서 정리한다.

## 2. 기본 사용 흐름
1) CSV 업로드 또는 직접 입력으로 참여자 목록 준비
2) 산정식 설정(가중치/제한) 확인 또는 조정
3) 랜덤 배정 실행
4) 데이터 검증(옵션)
5) 저장

## 3. CSV 업로드 안내
### 3.1 CSV 컬럼 순서
1) 참여자 성명
2) 참여유형
3) 성별
4) 생년월일
5) 모집경로
6) 경력유무(숫자)
7) 학력
8) 특정계층여부
9) 진행단계
10) 출장상담사

### 3.2 입력 규칙
- 성별: 남/여
- 생년월일: YYYY-MM-DD
- 참여유형: 1, 2, 1유형, 2유형
- 특정계층여부: O/X/Y/N/해당/해당 없음
- 출장상담사: 선택 입력

### 3.3 출장상담사 입력 시
- 해당 참여자는 입력한 상담사에게 고정 배정된다.

## 4. 직접 입력(수동 입력)
- CSV 없이 1건씩 추가 입력 가능
- 입력 후 “목록에 추가” 버튼을 눌러 하단 목록에 반영
- 출장상담사 선택 시 고정 배정 처리

## 5. 배정 결과 화면
### 5.1 상담사 통계 테이블
- 상담사별 배정 통계가 표시된다.
- 표의 “유형1/유형2” 값은 일일업무보고 히스토리로 저장된다.

### 5.2 엑셀 다운로드
- 배정 결과를 엑셀로 내려받을 수 있다.

## 6. 산정식 설정(가중치) 의미
가중치는 “어떤 기준을 더 중요하게 볼지” 결정한다.

- C_load: 현재 담당 건수가 많은 상담사에게 더 배정되지 않도록 조정
- C_fair: 유형/성별/연령대/특정계층 분포를 균형 있게 맞추는 정도
- C_streak: 연속 배정이 많지 않도록 조정
- C_pace: 최근 기간/연간 배정 편차를 줄이도록 조정

### 6.1 제한값
- Gap 임계값: 근속 그룹 내 편차가 커질 때 적게 배정된 상담사에게 우선 배정
- 일일 제한: 하루 배정 가능한 최대 건수
- G1/G2/G3 제한: 근속기간 그룹별 7/14/30일 배정 제한

### 6.2 랜덤 배정 계산식(요약)
상담사별 점수를 계산해 **점수가 가장 낮은 상담사**에게 배정한다.

```
총점 = (C_load * 가중치로드)
     + (C_fair * 가중치공정)
     + (C_streak * 가중치연속)
     + (C_pace * 가중치페이스)
```

- C_load: 현재 담당 건수와 같은 근속 그룹 평균의 차이가 클수록 점수 증가
- C_fair: 참여유형/성별/연령대/특정계층 분포가 균형에서 벗어날수록 점수 증가
- C_streak: 연속 배정이 길수록 점수 증가
- C_pace: 연간 배정 건수와 그룹 평균의 차이가 클수록 점수 증가

즉, **점수가 낮은 상담사**가 우선 배정되며, 가중치가 높을수록 해당 기준의 영향이 커진다.

## 7. 배정 방식 요약
- 출장상담사 입력이 있으면 해당 상담사로 고정 배정
- 나머지는 설정된 가중치/제한 규칙에 따라 자동 배정
- 배정이 불가능한 경우 미배정으로 남을 수 있다.

## 8. 저장 시 함께 기록되는 내용
- 배정 결과
- 일일업무보고(유형1/유형2)
- 산정식 설정값
- CSV 전체 입력 내용

## 9. 자주 묻는 질문
Q. 배정이 안 된 참여자가 있으면?
A. 저장 전 경고가 표시되며, 확인 후 진행 여부를 선택할 수 있다.

Q. 산정식 설정을 바꾸면 언제 적용되나?
A. 다음 “랜덤 배정” 실행 시 적용된다.
